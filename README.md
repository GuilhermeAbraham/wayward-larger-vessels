# Larger Vessels

A Wayward mod that increases the weight capacity of boat vessels, allowing you to carry more cargo.

## Overview

This mod enhances the storage capacity of all three vessel types in Wayward, making ocean exploration and transportation more efficient. The mod directly modifies both item and doodad descriptions to increase their weight capacities.

## Vessel Capacities

| Vessel Type | New Capacity |
|-------------|--------------|
| Raft        | 300          |
| Bull Boat   | 700          |
| Sailboat    | 1200         |

## Requirements

- **Wayward Version**: 2.15.1-beta
- **Game Mode**: Works in both single player and multiplayer

## Development

### Building from Source

Refer to [Wayward](https://github.com/WaywardGame) repository to confirm updated Mod updating instructions.

You can use the bellow commands if the game version is already updated.

```bash
# Install dependencies
npm install

# Build the mod
npm run build

# Clean build artifacts
npm run clean
```
